# Selfhost
Репозиторий с конфигурациями для запуска сервисов на собственном сервере.

## Устройства
На текущий момент следующие устройства работают внутри моей сети:
- Raspberry Pi 5 (8GB RAM)

## Сервисы
- [Uptime Kuma](https://uptime.kuma.pet/): Для отслеживания доступности сервисов и их аптайма;
- [Pihole](https://pi-hole.net/): Для блокировки рекламы и трекеров на уровне сети;
